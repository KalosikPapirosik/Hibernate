# Hibernate


Для запуска приложения необходимы Java 17 или выше, Maven и установленная СУБД PostgreSQL. Перед первым запуском необходимо создать базу данных lab-camp-psql, применить SQL-скрипт создания таблиц и указать параметры подключения в persistence.xml. Полною версию приложения можно найти на сайте GitHub по ссылке 
После запуска приложение инициализирует EntityManagerFactory, заполняет базу демонстрационными данными и открывает главное меню. Главное меню содержит раздел CRUD-операций, расширенного поиска, бизнес-запросов, заполнения базы демонстрационными данными и выхода.
В разделе CRUD пользователь может создать запись, просмотреть все записи, обновить отдельные поля и удалить выбранную сущность. В разделе бизнес-запросов пользователь может выполнить один из аналитических запросов или запустить все запросы последовательно.
